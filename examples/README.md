# Examples

This directory contains runnable Terraform/OpenTofu examples for `terraform-oci-fk-lpg`.

| Example | Description |
|--------|-------------|
| `01-basic-local-peering` | Two OCI VCNs connected with LPGs, with routing modeled through `terraform-oci-fk-vcn` |

The example is intentionally focused:
- `01-basic-local-peering` shows same-region LPG connectivity composed with reusable VCN modules.
